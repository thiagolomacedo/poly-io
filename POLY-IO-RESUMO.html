Poly.io

Chat sem barreiras de idioma

Sobre o Projeto

O Poly.io é uma plataforma de chat com tradução automática em tempo real. Cada usuário escreve mensagens no seu idioma nativo, e o destinatário recebe automaticamente a mensagem traduzida para o idioma configurado no seu perfil.

A tradução é invisível para o usuário final, criando uma experiência fluida de comunicação entre pessoas de diferentes países e idiomas.

Funcionalidades

Funcionalidade Descrição
Tradução Automática Mensagens traduzidas em tempo real para o idioma do destinatário
Detecção de Idioma Identifica automaticamente o idioma da mensagem enviada
Chat em Tempo Real Mensagens entregues instantaneamente via WebSocket
Ver Original Botão para visualizar o texto original da mensagem
Status Online Lista apenas usuários que estão conectados no momento
Seleção de Idioma Usuário escolhe seu idioma ao entrar no chat
Interface Responsiva Funciona em desktop e dispositivos móveis

Fluxo de Funcionamento

Passo Ação Resultado
1 Usuário A entra no chat Escolhe nome e idioma (ex: Português)
2 Usuário B entra no chat Escolhe nome e idioma (ex: Inglês)
3 A envia: "Olá, tudo bem?" Backend detecta: Português
4 Backend traduz Português → Inglês
5 B recebe "Hello, how are you?"
6 B pode ver original Botão "Ver original"
